About: This course introduces numerical methods (applied math) to non Computer Engineering / non Applied Math majors. This course covers taylor series, numerical differentiation, integration, root finding, linear algebraic equations, curve fitting, ordinary differential equations, and partial differential equations. The focus is on application of tools rather than theory. These tools are illustrated using Octave (a free software which has a workflow similar to MATLAB).
